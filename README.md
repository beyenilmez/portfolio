# portfolio

Personal site at [bedirhanyenilmez.com](https://bedirhanyenilmez.com).

Built with Vite, React 19, TypeScript, and CSS Modules. Bilingual EN/TR with static prerendering.

## Setup

```bash
pnpm install
pnpm dev        # Start dev server
pnpm build      # Production build
pnpm preview    # Preview built site
```

## Configuration

Site content is driven from `src/config/`. To update the site, edit these files, no component changes needed.

| File          | Purpose                                              |
| ------------- | ---------------------------------------------------- |
| `content.ts`  | Text, projects, experience, education, links (EN/TR) |
| `features.ts` | Animation toggles and thresholds                     |
